Förköps information

Vi säljer våra biljetter främst kontant via Österåkers kommun ungdomsgårdar. Du kan hitta gårdarna på Fritidsgardarna.se, och sedan kan du köpa biljetter via PayPal om du inte har möjlighet till att köpa kontant.
Glöm inte att innan du köper biljett läsa om våra regler. Detta kan du läsa i våran FAQ - Här


Köp kontant

Biljetter kostar 300 kr och kan köpas kontant via fritidsgårdarna i Österåkers kommun, vid köp måste du uppge ditt namn så att vi kan lägga till det på biljett nummret ifall någonting skulle hända. Sedan är du klar, du har en biljett till LAN:et!


Köp online

Det är enkelt att köpa online, vi erbjuder betalningar via PayPal. PayPal anses vara ett av de mest säkra betalsystemen online.
För att betala med paypal krävs det inte att du har ett konto, utan du kan köpa direkt via ditt kort direkt från PayPals hemsida. För att köpa via paypal kan du klicka på ovanstående bild.
